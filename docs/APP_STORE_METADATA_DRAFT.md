# App Store 메타데이터 초안 (ClassicLaunch)

## 1) 앱 이름
- ClassicLaunch

## 2) 부제(Subtitle, 30자 이내)
- 그리드형 클래식 앱 런처 복원

## 3) 프로모션 문구 (170자)
macOS 업그레이드 이후 그리드형 앱 실행 흐름이 그리웠다면, ClassicLaunch로 익숙한 런처 경험을 다시 만나보세요.

## 4) 설명(초안)
ClassicLaunch는 빠르게 앱을 찾고 정리하던 고전적인 그리드 런처 경험을 다시 제공하는 macOS 앱입니다.

주요 기능
- 전체 앱 자동 인덱싱
- 페이지 기반 그리드 레이아웃
- 실시간 검색 (앱 이름/경로/번들 ID)
- 폴더 생성/해제/이름 변경
- 드래그앤드롭 재정렬
- 클릭 즉시 실행
- 전역 토글 단축키(⌥⌘L)

개인정보 및 보안
- 앱 목록/배치 정보는 이 Mac 로컬에만 저장됩니다.
- 계정 생성이나 외부 서버 전송이 없습니다.

## 5) 키워드 (100자 내)
launcher,launchpad,app launcher,mac launcher,grid launcher,folder,productivity

## 6) 카테고리
- Productivity

## 7) 연령 등급 가이드
- 4+

## 8) 리뷰 노트(심사용)
ClassicLaunch는 grid-based app launcher experience를 제공하는 독립 앱이며,
private API를 사용하지 않습니다. 모든 데이터는 로컬 저장소에만 보관됩니다.

## 9) 스크린샷 제안
1. 메인 그리드 + 페이지 인디케이터
2. 폴더 열기/편집 화면
3. 검색 오버레이
4. 드래그 정렬 장면
5. 개인정보 안내(온디바이스 처리) 노출 장면
